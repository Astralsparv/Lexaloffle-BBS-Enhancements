# Lexaloffle-BBS-Enhancements
Enhancements for the Lexaloffle BBS

Not affiliated with Lexaloffle.

## Installation
### Chromium users:
It costs money to put it on the chromium web store which I will not be paying for.
Go to `chrome://extensions` and enable Developer Mode
Drag and drop the downloaded extension (this repository) into it.
Enable it!

### Firefox support not included *yet*.
